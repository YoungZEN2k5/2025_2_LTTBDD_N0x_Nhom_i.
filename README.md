# To-Do Bloom (Quản lý công việc cá nhân)

Ứng dụng mẫu quản lý công việc cá nhân viết bằng Flutter. Ứng dụng hỗ trợ:

- Tạo / xóa / đánh dấu hoàn thành công việc
- Phân loại theo danh mục
- Lọc theo ngày và xem lịch
- Giao diện song ngữ (Tiếng Việt / English)
- Chế độ giao diện sáng / tối
